# 《LLM 디자인 패턴》(가제) 코드

## 코드에 관해

2~10장 코드(`.md`)는 원서 출판사의 깃허브 저장소에서 가져온 것입니다.

**주의**: 실제 작동하는 코드가 아니라 개념 이해를 돕기 위한 placeholder입니다.

역자가 테스트한 일부 장의 노트북(`.ipynb`)이 [notebooks](notebooks) 폴더에 있습니다.
